## 🚀 Task 1: Stock Portfolio Tracker

### Features
- 📈 Track multiple stocks with real-time prices
- ➕ Add stocks with custom quantities
- ❌ Remove stocks from portfolio
- 💰 Automatic total portfolio value calculation
- 📊 Weight percentage display for each holding
- 💾 Local storage persistence (data saves automatically)
- 📥 Export portfolio to CSV file
- 📱 Responsive design for mobile & desktop

### Technologies Used
- HTML5
- CSS3 (Grid, Flexbox)
- Vanilla JavaScript (ES6)
- Local Storage API

### How to Use
1. Open `Task1_Stock_Portfolio/stock_portfolio_tracker.html`
2. Click "Add / Remove" tab
3. Select a stock and enter quantity
4. Click "Add to portfolio"
5. View your portfolio in the "Portfolio" tab
6. Export data using the "Export CSV" button

## 🧮 Task 2: Interactive Calculator

### Features
- ➕ Addition, subtraction, multiplication, division
- ⌨️ Full keyboard support:
  - Numbers (0-9)
  - Operators (+, -, *, /)
  - Enter key for equals (=)
  - Escape key for AC (All Clear)
  - Backspace for delete
- 🔄 Chain calculations
- 🎨 Modern dark theme UI
- ⚡ Real-time display updates
- ⚠️ Error handling (division by zero)
- 📱 Responsive design

### Technologies Used
- HTML5
- CSS3 (CSS Grid, Custom Properties, Animations)
- Vanilla JavaScript (State management)

### How to Use
1. Open `Task2/calculator/index.html`
2. Click buttons or use keyboard
3. Press "AC" to clear all
4. Press "C" to delete last character
5. Press "=" or Enter to calculate

### Keyboard Shortcuts
| Key | Action |
|-----|--------|
| 0-9 | Enter numbers |
| + - * / | Operators |
| Enter or = | Calculate |
| Escape | Clear all (AC) |
| Backspace | Delete last |
| . | Decimal point |
| % | Percentage |
